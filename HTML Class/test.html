CSS Value Functions

Complete Guide with Practical Examples

Introduction to CSS Value Functions

Note: CSS value functions are special operations that accept parameters and return values that can be used as CSS property values. They enable dynamic calculations and transformations directly in CSS.

CSS functions allow you to set property values dynamically, perform calculations, transform elements, and create complex effects without JavaScript.

Mathematical Functions

calc() Function

The calc() function performs calculations to determine CSS property values.

Basic calc() Example

Width: calc(100% - 100px)
.calc-box { width: calc(100% - 100px); height: 80px; background: var(--primary-color); }

min() and max() Functions

The min() and max() functions let you set a value as the minimum or maximum of a set of values.

min() Function Example

Width: min(600px, 90%)
.minmax-box { width: min(600px, 90%); height: 80px; background: var(--secondary-color); }

clamp() Function

The clamp() function clamps a value between an upper and lower bound.

clamp() Function Example

Font-size: clamp(1.5rem, 4vw, 3rem)
.clamp-box { font-size: clamp(1.5rem, 4vw, 3rem); background: var(--accent-color); padding: 20px; }

Transform Functions

Note: Transform functions modify the coordinate space of an element, allowing rotation, scaling, skewing, and translation.

translate() Function

translate(20px, 10px)
.box { transform: translate(20px, 10px); }

rotate() Function

rotate(15deg)
.box { transform: rotate(15deg); }

scale() Function

scale(1.2)
.box { transform: scale(1.2); }

skew() Function

skew(15deg, 5deg)
.box { transform: skew(15deg, 5deg); }

Combining Transform Functions

Combined Transforms
.box { transform: translate(30px, 20px) rotate(10deg) scale(1.1); }

Gradient Functions

Note: Gradient functions create progressive transitions between two or more colors. CSS supports linear, radial, and conic gradients.

linear-gradient()

.gradient-box { background: linear-gradient(45deg, #3498db, #2ecc71); }

radial-gradient()

.gradient-box { background: radial-gradient(circle, #3498db, #2ecc71); }

conic-gradient()

.gradient-box { background: conic-gradient(#3498db, #2ecc71, #e74c3c, #3498db); }

Filter Functions

Note: Filter functions apply graphical effects like blurring or color shifting to elements. Filters are commonly used with the filter property.

blur()

.filter-box { filter: blur(3px); }

grayscale()

.filter-box { filter: grayscale(100%); }

brightness() and contrast()

.filter-box { filter: brightness(1.5) contrast(1.2); }

Multiple Filters

.filter-box { filter: blur(2px) grayscale(50%) brightness(1.2); }

Color Functions

Note: CSS provides several functions for working with colors, including rgba(), hsla(), and the newer color() function.

rgba()

rgba(52, 152, 219, 0.7)
.box { background: rgba(52, 152, 219, 0.7); }

hsla()

hsla(204, 70%, 53%, 0.7)
.box { background: hsla(204, 70%, 53%, 0.7); }
